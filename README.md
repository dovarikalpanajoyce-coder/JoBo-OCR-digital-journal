## Overview
JoBo is a digital journaling application that uses Optical Character Recognition (OCR) to extract text from handwritten and printed documents. The system converts unstructured image data into searchable and editable digital text.

## Features
- OCR-based text extraction
- Image preprocessing for improved accuracy
- Secure storage of extracted journal entries
- AI-driven text processing

## Technologies Used
- Python
- OpenCV
- Tesseract OCR
- NumPy
- Pandas

## How It Works
1. User uploads an image or scanned document
2. Image preprocessing is applied
3. OCR engine extracts text
4. Extracted text is stored and displayed digitally

## Use Cases
- Personal journaling
- Digital note archiving
- Text digitisation from handwritten documents

## Future Improvements
- NLP-based sentiment analysis
- Cloud deployment
- Multi-language OCR support
